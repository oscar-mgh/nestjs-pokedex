# Ejecutar en desarrollo

1.- Clonar el repositorio

2.- Ejecutar
```
yarn install
```

3.- Instalar Nest CLI
```
npm install -g @nestjs/cli
```
4.- Servir la base de datos
```
docker compose up -d
```

## Tech Stack:
* MongoDB
* Nest
